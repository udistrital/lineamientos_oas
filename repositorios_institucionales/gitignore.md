# Gitignore

En está sección se establecerán las convenciones para la definción del archivo **.gitignore** de los repositorios para la Oficina Asesora de las Tecnologías y la Información (OATI).

## Proyectos Beego

  ```javascript
  lastupdate.tmp
  .vscode/
  titan_api_crud
  main
  .scannerwork/
  swagger/**
  !swagger/swagger.json
  !swagger/swagger.yml
  ```

- **.vscode/**
- **lastupdate.tmp**: Archivo generado por el Api beego
- **titan_api_crud** Es la compilado del Api que genera Beego
- **main**: Es la Compilación del archivo main.go
- **.scannerwork/**: Es el directorio que genera el sonar-scanner
- **swagger**: Del directorio swagger solo se requieren los archivos swagger.json y swagger.yml
